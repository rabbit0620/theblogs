# 缺省逻辑

Parent: [ai_keywords]({{< ref "ai_keywords" >}})

### 缺省逻辑（Default Logic）

#### 核心定义  
缺省逻辑由Raymond Reiter于1980年提出，是一种**非单调逻辑**形式化框架，旨在处理信息不完全场景下的推理。其核心机制是**缺省规则**（Default Rule）：当已知条件成立且某个结论与当前知识库一致（无矛盾证据）时，默认该结论为真；一旦新证据出现导致矛盾，则撤销该结论。这是经典单调逻辑（如一阶逻辑）无法覆盖的关键特性。

#### 关键技术点  
1. **缺省规则**：形如 `α : β / γ`，其中前提`α`成立，且`β`（正当性条件）与当前知识**无矛盾**，则默认得出推论`γ`。  
2. **扩展（Extension）**：由缺省规则集和事实集生成的最小不动点。一个理论可能有**多个扩展**，代表不同合理假设分支，需通过优先级（如优先缺省逻辑）或证据权重选择最可靠者。  
3. **非单调性**：新知识的加入可能使已默认推出的结论失效，这与人类认知中的“常识推理”高度一致。  
4. **封闭世界假设（CWA）区分**：CWA假设未证明为真者皆假（如数据库）；缺省逻辑仅在无矛盾时默认真，保留撤销弹性。  
5. **语义与计算**：基于极小冲突集、优先序或弱化扩展的语义变体（如不连续缺省逻辑），解决一致性和计算复杂度（典型问题为NP-hard）。

#### 医学/神经科学应用场景  
以首都医科大学神经病学重点实验室的**阿尔茨海默病（AD）早期智能诊断**为例：  
- **背景**：AD诊断依赖认知量表、脑脊液生物标志物（Aβ、p-Tau）及影像数据，但早期症状不典型或检测结果部分缺失。  
- **缺省逻辑建模**：定义缺省规则如“若患者记忆下降且无脑血管事件证据，则默认其海马体存在萎缩（MCI阶段）”。该规则可生成多个扩展（如血管性痴呆、抑郁假性痴呆的排除）。  
- **推理流程**：系统初期凭缺省逻辑输出可能病理谱；当后续淀粉样PET扫描呈阳性时，规则自动否决先前假设，转向AD诊断扩展。  
- **临床价值**：模拟神经内科医生的“试探性诊断-修正”思维，在数据不完整时提供可解释的共识输出，显著提升早期诊断召回率。
