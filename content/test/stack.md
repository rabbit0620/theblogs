所有Markdown语法测试文档

1. 标题

# 一级标题（H1）
## 二级标题（H2）
### 三级标题（H3）
#### 四级标题（H4）
##### 五级标题（H5）
###### 六级标题（H6）


2. 段落与换行

这是普通段落。
这是换行后的内容（行尾加两个空格+回车）。

空行会分隔段落。


3. 文本样式

*斜体* 或 _斜体_
**粗体** 或 __粗体__
***粗斜体*** 或 ___粗斜体___
~~删除线~~
<u>下划线（部分解析器支持HTML）</u>


4. 列表

无序列表

- 项目1
- 项目2
  - 子项目2.1
  - 子项目2.2
* 项目3
+ 项目4


有序列表

1. 第一项
2. 第二项
   1. 子项2.1
   2. 子项2.2
3. 第三项


任务列表（GFM扩展）

- [ ] 未完成任务
- [x] 已完成任务
- [ ] 待办事项


5. 链接

行内链接：[百度](https://www.baidu.com "百度首页")
参考式链接：[GitHub][gh-ref]
[gh-ref]: https://github.com "GitHub官网"
自动链接：<https://www.google.com>


6. 图片

行内图片：![示例图片](https://picsum.photos/200/100 "随机图片")
参考式图片：![备用图片][img-ref]
[img-ref]: https://picsum.photos/300/150 "另一张随机图片"


7. 代码

行内代码

`print("Hello Markdown")`


围栏代码块（指定语言）

```python
def hello():
    print("Hello World!")
```


缩进代码块（4空格）

    function test() {
        console.log("Test");
    }


8. 表格（GFM扩展）

| 左对齐 | 居中对齐 | 右对齐 |
| :--- | :---: | ---: |
| 单元格1 | 单元格2 | 单元格3 |
| 数据A | 数据B | 数据C |


9. 引用

> 一级引用内容。
>> 嵌套的二级引用。
> 回到一级引用。


10. 分割线

***
---
___


11. 脚注（部分解析器支持）

这是一个脚注示例。
另一个脚注[^note]。

: 第一个脚注的内容。
[^note]: 第二个脚注的说明。


12. 数学公式（LaTeX，部分解析器支持）

行内公式：$E=mc^2$
块级公式：
$$
\sum_{i=1}^{n} i = \frac{n(n+1)}{2}
$$


13. HTML元素（部分解析器支持）

<div style="color: red;">红色文字</div>
<span style="font-weight: bold;">加粗内容</span>
<br>换行<br>
<hr>自定义分割线<hr>


14. 转义字符

\*这不是斜体\*
\#这不是标题
\[不是链接\]


注意事项

不同解析器（如GitHub Flavored Markdown、CommonMark、Typora等）对语法的支持存在差异，例如数学公式、脚注、任务列表等功能并非所有环境都兼容。

你可以将上述内容保存为.md文件，用不同工具打开测试效果~
